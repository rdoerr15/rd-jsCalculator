Instructions

When using the standard functions (add, subtract, multiply, and divide), enter inputs for the first and second number then click the appropriate button to generate the result. If you'd like to use the sum all, multiply all, minimum, maximum, or average functions, you'll need to enter a series of numbers in the corresponding input box, then click the appropriate button. The series of numbers will be read as all single digit integers, so there's no need to separate numbers with commas.
